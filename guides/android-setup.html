酒馆搭建教程——安卓手机

在 Android 手机上安装 Termux，一键部署酒馆，随身携带随时玩。

📄 原始教程来自飞书云文档 →
本教程默认你已看过「哪种搭建方式适合你？」，选择了安卓本地搭建的方案。全程需要挂好魔法，可以快非常多。教程使用 Termux + 钓鱼佬一键脚本，是目前最省心的安卓部署方式。
01 下载 Linux 模拟器（Termux）

Termux 是安卓上的 Linux 模拟器，酒馆就运行在它里面。

Termux 下载（GitHub Releases）

02 安装酒馆

类脑现在有很多一键脚本，推荐用钓鱼佬的工具箱：

钓鱼佬的工具箱 - 安卓酒馆部署脚本 →

把脚本代码粘贴到 Termux 的黑框里回车，等一通自动下载安装就完事了。

安装完成后，输入数字 1 回车启动酒馆。第一次启动会比较慢，之后浏览器会自动弹出 127.0.0.1:8000，没跳转就手动输入。

Termux 一定要在后台挂好！它才是酒馆的本体，浏览器那边只是个小窗口。Termux 关了什么都没有了。
03 MT 管理酒馆文件

下载 MT 管理器，最新版，用于快速访问酒馆文件做备份和修改：

MT 管理器下载

安装后打开，左上三横进设置 → 中上三个点 → 添加本地存储。（界面不一样就更新一下软件）

Termux 要先打开，MT 管理器才能看到 Termux 内的文件（酒馆）。找到 termux 授权访问即可。

MT 管理器的操作逻辑：左右各一个文件夹，复制剪切对比都很方便。在文件名上左右滑动选中（支持多选），长按弹出具体操作。

05 修改 config（电脑访问安卓酒馆）

Termux 要先打开，MT 管理器才能看到 termux 内的文件。

5.1 开监听 + 多账户

允许其他设备访问（改东西还是电脑爽），再开多账户（偶尔给朋友玩一下）：

listen: true whitelistMode: false basicAuthMode: true enableUserAccounts: true # 默认账户，不用改 basicAuthUser: username: user password: password

四个改完后，右上保存，重启酒馆。

打开手机 WIFI 设置，找到本地局域网 IP，电脑浏览器用这个 IP + :8000 就能访问手机酒馆了（如 192.168.1.100:8000）。这个 IP 在路由器重启后会变，跨设备访问出问题了先查这个。

5.2 设置多用户

登录后是默认的管理员账户，在顶部的「人与齿轮」（用户设置）处操作：

给默认管理员账户设置密码 → 添加新用户 → 退出登录 → 用新用户登录。

5.3 关基础验证

关掉 Termux 重开，再改 config.yaml（别忘了保存）：

basicAuthMode: false

把基础验证模式关掉后（登录页左边那个框消失），下次浏览器直接用刚才新设的用户密码登录就行（只剩右边的框）。
